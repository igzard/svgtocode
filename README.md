
# SVG to CODE

  

Change ```<img  src="xyz.svg">``` html tag to svg source code for change svg color with CSS

## Usage

```

<script src="jquery.min.js"></script>
<script src="jquery.svgtocode.min.js"></script>

```
  

## Getting Started

### JS

```

$('#elementID').SvgToCode();

```

### CSS

```

svg path {

	fill: COLOR;

}

```
